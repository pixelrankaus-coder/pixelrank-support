# Tailwind Rules for Pixel Rank CRM V2

1. Use this Tailwind config only:
   client/metronic/metronic-v9.3.6/metronic-tailwind-react-starter-kit/typescript/vite/tailwind.config.js

2. Do not create any new Tailwind config files.

3. Do not change Metronic colour tokens, fonts, or spacing scales without approval.

4. When building new components:
   • Use existing Metronic utility classes.
   • Avoid custom CSS unless there is no other option.

5. Never copy Tailwind or Webpack configuration from HTML kits, demos, or concepts into the main app.
